Algorithmic Function Library

Contains a number of functions for the purpose of solving math problems. Aids in a modular approach to problem solving. Contains examples of lambda expression

public delegate Tuple NextSequenceElement(int[] input)
Abstract Lambda Function that accepts an integer array as input and returns a tuple containing the next integer element of a recursive sequence and input array required
public static int SumArray(int[] array)
Sum the elements of an integer array
public static Tuple NextLesserPanlindrome(int[] input)
Function that takes an integer input array of the numbers of the currnet Panlindrome. Returns a Tuple containing the the next smallest Panlindrome in both integer and integer array format
public static char[] IntToCharArray(int num)
Convert an Integer to an Char array of its digits
public static int CountIntegerChar(int num)
Count the number of digits in an Integer
public static List MakeSequence(NextSequenceElement generateNext, int first, int length, params int[] inputs)
Creates a recursively defined sequence of a specified length in the form of a list of integers.
public static List MakeSequenceUnder(NextSequenceElement generateNext, int first, int max, params int[] inputs)
Creates a recursively defined sequence in the form of a list containing all element under a specified number given an starting number
public static Tuple NextEvenFibonacci(int[] input)
public static List ListPrimes(int num)
find a list of num primes using the sieve of Erotoshenes
public static List ListPrimesUnder(int num)
find a list of all positive primes under num using the sieve of Erotoshenes
public static List PrimeFactorize(long number)
returns a list of size 2 int arrays with the 1st element being the prime number and the 2nd its power

//Returns a list of all possible multiples of specified variables up to max
